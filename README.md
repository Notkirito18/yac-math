# yac-math

my little math library

## basic usage

- ```npm install yac-math```

### available methods

#### returns boolean

- ```isEven( value )```
- ```isNaturalNumber( value )```
- ```isPrimeNumber( value )```

### importing

```javascript

    const yacMath = require( 'yac-math' );
    // OR
    const {isEven, isNaturalNumber, isPrimeNumber} = require( 'yac-math' );


```
